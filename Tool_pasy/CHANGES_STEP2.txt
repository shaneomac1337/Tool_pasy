================================================================================
MATCHING ALGORITHM
================================================================================

Input: Plant name from invoice (e.g., "Cephalotaxus fortunei 60-80 cm")

Process:
1. Normalize: lowercase, trim, remove dimensions
   → "cephalotaxus fortunei"

2. Exact match against all normalized sarze names
   → If found: return match_type='exact', confidence=100%

3. Fuzzy match using token_sort_ratio (theFuzz library)
   → If score >= 65%: return match_type='fuzzy', confidence=score%
   → Else: continue

4. No match found
   → return match_type='none', confidence=0%

Configuration:
  EXACT_THRESHOLD = 95   (currently unused, all matches are 100% or fuzzy)
  FUZZY_THRESHOLD = 65   (minimum score to suggest fuzzy match)
